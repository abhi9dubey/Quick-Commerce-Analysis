# Quick-Commmerce Data Analysis Dashboard by Abhinav Dubey

## Project Overview

This project presents a Power BI dashboard that analyzes Blinkit's sales performance, customer satisfaction, and inventory distribution.  
It highlights key insights and opportunities for improvement using KPIs and visualizations.

## Key Performance Indicators (KPIs)

The dashboard tracks these main KPIs:

1. **Total Sales**: Total revenue from all items sold ($1.20M)
2. **Average Sales**: Average revenue per sale ($141)
3. **Number of Items**: Total number of different items sold (8523)
4. **Average Rating**: Average customer rating for items sold (3.9 out of 5)

## Features

- **Filter Panel**: Filter data by outlet location type, outlet size, and item type  
- **Outlet Establishment Trend**: Shows the growth of outlets from 2012 to 2022  
- **Fat Content Analysis**: Compares sales of low-fat and regular-fat products  
- **Item Type Distribution**: Displays sales across various product categories  
- **Outlet Size and Location Analysis**: Highlights sales by outlet size and location tier  
- **Outlet Type Comparison**: Compares outlet types by sales, number of items, average sales, ratings, and visibility  

## Insights and Conclusions

1. Strong overall sales with more than $1M in total revenue  
2. Consumers prefer low-fat products, showing a focus on healthy choices  
3. Fruits, vegetables, and snacks are the top-selling categories  
4. Medium-sized outlets in Tier 3 locations are the most profitable  
5. Supermarkets make higher sales, while grocery stores have better product visibility
